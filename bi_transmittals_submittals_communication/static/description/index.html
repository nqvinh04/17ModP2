Submittals-Transmittals Document for Job Contracting and Construction Odoo Apps

Job Order Construction Transmittals and Submittals Documents Odoo Apps helps to Manage Transmittals / Submittals Document of job order-construction and Job Costing Contracting. After installing this odoo module user can create different document for construction projects with different document type and able to submit and transmit to the whole contracting process.

This odoo apps is use for complete document management integration with job order construction and contracting process. User can choose different medium for submission of the generated documents and its link with job order, job cost sheet, project different view where you can use this feature to transmit and submit generated documents.
