Manufacturing Secondary Finished goods - Several Outputs in MRP Odoo Apps Community Edition

Manufacturing Several Outputs/ Secondary Products in MRP odoo apps helps user to produce secondary product alongside with finished products in manufacturing order also it allows to finished manufacturing order with different quantity then plan quality of production order with different secondary outputs as configured on bill of materials. First production user can configured BOM with secondary product outputs which means this product will come as secondary by-products when manufacturing order will finish.

This odoo apps mostly use for food manufacturing industry where user don't know what will be actual finished good quantity after finished the manufacturing order and finished goods comes with several different outputs/secondary finished goods. By default Odoo doesn't allow to record production for more or less than plan produce quantity with multiple finished goods which consumed planned materials. Our this odoo module solved that problem so production user can record produce quantity as per actual output they receive after that production along with secondary finished goods, which consumed material as per plan quantity . If you have multiple work order on manufacturing order then change qty option and secondary output product selection appears on the last work order screen so once you finished the record production it will logged added finished goods quantity and finished manufacturing order.

Scenario One

Bills of Material

User can see "Secondary" tab on bills of material form view, also add secondary product.

Manufacturing Order

Manufacturing Workorder

User have to click "Start Working" button on manufacturing workorder form view.

Validate Secondary Product

On work order processing user can enter more or less quantity then quantity to produce and also can see and enter produced secondary products quantity and click on "Done" button.

On manufacturing order under "Finished Products" tab user can see secondary products alongside with finished product.

Inventory Moves

User can see generated inventory move for manufactured product and secondary product.
