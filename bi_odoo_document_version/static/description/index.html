Automatic Attachment-Document Version and Versioning Management Odoo Apps

Auto Document Versioning management Odoo apps generate automatic document version on attachment records. After installing this odoo module whenever any same document has been generated from Odoo or upload from odoo it will automatically generate versioning-numbering on attachment in document management system-DMS i.e When any accountant print invoice multiple times generated and attached invoice pdf report automatically generate versioning numbering.

Features

Document Versions

Auto Version system for attachment record with the same name.

Document History

System user can track the documents next/previous versions.
Demo Request Video Tutorial

Attachment On Project Task

User can see multiple attached documents.

Document Kanban View

Under Settings > Technical > Attachments menu, user can see attached documents with version history in kanban view.

Attachment Tree View

Under Settings > Technical > Attachments menu, user can see attached documents with version history in tree view.

Document Versions/Document History

User can see "Document Version", "Document Previous Version", and "Document Next Version" on document form view.

Restrict Automatic Document Version

Under Settings > Technical > Restrict Attachments menu, user can create and select models to restrict automatic document version.

User can see document version will not created for attachments oof selected models.
