Equipment and Maintenance Request for Job Work Order Odoo Apps

Job Order Equipment MRO-Maintenance Request Odoo app allow you to manage Equipments and Equipment Maintenance Request from Job Work order in construction project/contract in Odoo. This odoo apps helps to integrate Equipment and Maintenance Request with project job costing construction.

Project Construction Job Costing and Job Cost Sheet Odoo app used to manage project job cost and job cost sheet with material request in construction management system in Odoo This Odoo module also help to calculate progress costing and cost calculation in each Construction project job cost with job order with Material Product Purchase Requisitions it use for product purchase requisition process by employee and users. Once purchase requisition placed for required goods and material, by default it will be place from the configured location for users and employee. Project Job cost sheet calculates complete cost of the each project including material, labour and overhead cost for material it's also generate stock movement when material is consume for the process. User can able to print different reports for this job costing apps.

Create Equipment Request From Job Order

Users can create equipment requests from job orders by clicking the "Add a line" button under the "Equipment Request" tab.

Equipment Request Wizard

On the equipment request wizard, the user can select equipment machine, equipment category, and other details and click the "SAVE" button, User also can click the "START" button to start equipment request.

Create Equipment

Users can create equipment from job order by clicking the "CREATE EQUIPMENT" button under the "Equipment" tab.

Create Equipment Wizard

Users can enter equipment name, category, model, and serial number and click the "CREATE EQUIPMENT" button.

Users can see created equipment added to the job order.

Smart Buttons

Users can see smart buttons on the job order view for easy access and redirect to related equipment and its requests.
