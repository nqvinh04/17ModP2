Work Order - Job Order with Start and Stop Timer and Digital Signature Odoo Apps

Work Order Start and Stop Timer With Digital Signature Odoo apps help employees to set Duration By buttons and it will allowed to sign on task after completing job order. This Odoo apps added to start and stop timer on job order for project costing so when user started the work order they can start the timer and once they stop work order they can stop the timer that's how real time duration will be calculated for work order. Also it has digital signature feature on job costing apps so user can sign job order document using digital signature.

Features

Timer

Allow your Employee to start/stop/pause clock timer on job order form of website portal my account page.

Signature

Digital signataure will be store in backend job order record.
