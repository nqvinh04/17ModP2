Warehouse Stock Transfer Backdate and Remarks Odoo Apps

This Odoo apps help to put custom backdate and remarks on inventory transfer and stock transfer and scrap order. Those odoo modules allow inventory backdating operations in Odoo warehouse stock transfer option, backdate will be transfer to stock entries and accounting entries automatically when you deliver the goods to customer while process delivery orders. Whatever remarks you want to add you can add those custom remarks on delivery order and this remarks automatically pass on stock accounting journal entry if you set inventory valuation method as Automated. You can also update mass backdate for multiple transfers with single click.

Warehouse Stock Transfer Backdate and Remarks

Validate Transfer

User have to click on validate button on stock transfer.

Process Backdate and Remarks Wizard

User need to enter "Transfer Date" and "Remark" for stock transfer and this will pass to stock move, product move and journal entry.

Transfer Date and Remarks on Stock Transfer

User can see entered transfer date and remarks added to stock transfer.

Transfer Date & Remarks on Stock Moves

User can see transfer date and remarks added to stock move created for stock transfer.

Transfer Date on Journal Entry

User can see transfer date added to journal entry created for stock transfer.

Update Mass Backdate on Transfers

User can see select multiple stock transfer from tree view, and under "Action" menu user can see "Update Mass Backdate" menu to update backdate and remarks for multiple selected stock transfer backdate and remarks.

User can enter backdate and remarks for selected multple transfers and this will added to transfer, stock moves, product moves and journal entry for that.

Updated Mass Backdate and remarks on Transfers

User can see entered transfer date and remarks added to stock transfer.

Transfer date and remarks also added to stock moves, product moves and journal entry for all selected multiple transfer.

Scrap Order Transfer Backdate and Remarks

Validate Scrap Order

User have to click on "VALIDATE" button on scrap order.

While validating scrap order user can enter "Backdate" and "Remarks" for scrap order and this will pass to product move.

Backdate and Remarks on Scrap Order

User can see entered backdate and remarks added to scrap order.

Transfer Date on Product Moves

User can see transfer date added to product move created for scrap order.
