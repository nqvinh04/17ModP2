Construction Projects Details on Website Odoo Apps

This Odoo Apps is used to view your construction projects on your website under Project menu, where you can manage your contractors, agencies, vendors in Odoo ERP. This Odoo module helps to show Construction Projects details on website which shows project categories, project description, construction Project brouchers, Project Construction images, Project Specification, location of construction project site with contact details etc in website with this Odoo app.
