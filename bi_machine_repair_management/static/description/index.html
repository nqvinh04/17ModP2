All Types Machine-Product Repair Management with Website Odoo Apps

Machine-Product repair management Odoo Apps helps to manage repair order process and workflow for any kind of machine and product to Manufacturers and industrial-supply companies. After installing this odoo module user can also have option to register machine repair request from the website and portal also from the Backend. After the repair order submitted technician can start diagnosis problem of the machine, able to fill out timesheet for the spent hours. Technical can able to create workorder for different repair process and service manager will create quotation based on required consumed parts and timesheet and complete flow of repair process. This Maintenance repair Odoo apps also support different pdf reports repair order report, label etc.

Features

Order repair request from portal.

Customer can send machine repair request from website and also attach documents.

Create Quotation for machine repair diagnosis.

According to the result of diagnosis quotation will be created and send to the customer.

Manager can create/manage machine repair request and can assign Machine repair request to technician.

After receiving repair order, manager will diagnosis the repair order and enter result into it.

Report Generation.

PDF reports for machine repair request.

Manager/Technician can create diagnosis based on machine repair request.

On reveiving diagnosis report worker/technician will work on that.

Various Configuration..

Configure Machine and Machine Parts also Configure Machine Repair Teams, Services, Service Types.
