Sales Contract/ Subscription Management and Recurring Invoice

This module allow you to create product for warranty/maintenance/subscription and based on that product on Sales order line it will create sales contract for that customer automatically on confirmation of sales order. It is also allowing you to create recurring invoice from contract based on period configured on contract. It also support in future if you sale some more service on same contract then you can select that contract on quote/sales order and it will append that line on contract.

Main Features:

  • Allow you to create a product for subscription so products which are set as subscription products then system will automatically create contract/subscription when sales order of that product is confirmed.
  • Allow your sales team and accounting team to work on Subscription/Contract created from Sales order.
  • Details of Sales order and Sales order lines will be available on Subscription/Contract form.
  • Subscription lines will be available on Subscription form which are created from Sales order.
  • Allow you to set up recurring periods on Subscription and invoice based on that.
  • Quick generate customer invoice from Subscription(s) form.
  • Cron Job to automate invoice creation process by system until Subscription reached its end date.
  • Mass customer invoice generation facility for Subscription(s).
  • Manual Process to Generate Invoice directly from Contract menu and Generate Invoice Button.
  • Manual Process to Generate Recurring Invoices wizard in single click.
  • Automatic create Recurring Customer Invoice/Bill using cron job for Subscription/Contract.
  • Print Subscription/Contract in PDF format.
  • If Subscription / Contract on Sales order then system will just append new Subscription lines inside that and will not create new Subscription for that sales order.
  • for more details please see below screenshots and watch the video.

Process Options of Sales Contract/ Subscription Management and Recurring Invoice:

Configuration:

  • Configure product as Contract/Warranty/Subscription Product under Products menu.
  • Configure Quotation Template and Template Lines under Configuration/Quotation Template Menu.

    • Process:

  • There are two process of Contract recurring invoice analytic.

    Option 1. Create New Contract/Subscription from Quote / Sale Order
    Option 2. Update Existing Contract/Subscription from Quote / Sale Order

    • Option 1. Create New Contract/Subscription from Quote / Sale Order:

  • Allow you to create new contract once sales order will be confirmed.
  • Created Contract will appear in Contract menu.
  • Workflow on Contract Form.
  • Contract will show contract lines and sales line.

    • Option 2. Update Existing Contract/Subscription from Quote / Sale Order

  • Allow you to update existing contract which have been selected on sales order during confirm so all subscription product and its update will be recorded on existing contract/subscription lines and if there is new product in this sales order it will be added as new contract lines.
  • Workflow on Contract Form.
  • Contract will show contract lines and sales line.

    • Process (Options) of Generate Invoice:

  • Option 1. Manually Generate invoice from Generate invoice button on contract/subscription form.
  • Option 2. Automatic Create Customer Recurring Invoice/Bill using Cron Job.
  • Option 3. Manual Process to Generate Recurring Invoices using wizard for multiple contracts/subscriptions.

    • Reports:

  • Print Contract/Subscription PDF Report.

    • For more features/apps you can also see full bundle: https://apps.odoo.com/apps/modules/17.0/subscription_recurring_contract_bundle/

    Product Configuration

    Create product as Contract/Warranty/Subscription product and set Recurring Period.

    Quotation Template

    Create your custom quotation template using contract/subscription product this will allow you to select on sale order and auto set contract/subscription line on sales order.

    Case 1 - Sales Quote / Sale Order

    Create Quote by selecting Quotation template and that will set contract/warranty line on sale order lines along with selling product lines.
    We have to select recurring period on sales order which will be used during recurring invoice creation from contract.

    Sale Order Confirmation

    If you have not selected Analytic Account / Contract on Sales order then it will create new Contract / Analytic Account for that customer.
    Module also support like if you have selected existing Contract/ Analytic Account then it will append Subscription/Contract line into existing contract. (For new subscription product it will append in contract but for existing subscription product it will increase qty in contract.)

    Contract from Sale Order Confirmation

    We have separated Contract/Subscription lines and Sellable Product lines.
    You can find new fields on contract/analytic account as below which will allow you to manage customer contract and recurrning invoice.

    • Recurring Period: Invoice creation period.
    • Contract Expiration Days: Set contract stage set to "Expires Soon" before end date.
    • Date Of Next Invoice: Next invoice will create on this date.
    • Order/Invoice Currency: Invoice will create in this currency and this currency is fetch from Sale Order
    • Stage: Contract Status (New->Running->Expires Soon->Expired->Locked)

    Selling Product On Contract

    Using selling product you can have idea for this contract which product originally sold with qty sold and pricing details. And it can be used in future.

    Recurring Invoice from Contract

    Create Recurring Invoice from Contract using Generate Invoice button on contract form. And Invoice will be created with only contract/warranty/subscription product lines in contract currency.

    Case 2 - Existing Contract On Sale Order

    If we have selected contract on Sale Order then it will merge/append contract/subcription product lines to contract subscription lines.

    Added Contract/Subcription Lines on Contract

    Updated Contract/Subcription Lines on Contract

    If we have same warranty/contract/subcription line on selected contract then it will increase product quantity otherwise it will create new contract/subscription lines

    Updated (Without Contract/Subcription Product) Sale Order Lines on Contract

    If we added Selling Product line on selected contract then it will updated or added product in Sale Order Lines.

    Automatic Create Customer Invoice/Bill using Cron Job

    Cron will search for all contract/subscriptions which has "Date of Next Invoice" is equal to Today date and create invoice for that contract.

    Manual Process to Generate Recurring Invoices Wizard in Single Click

    This wizard will search for all Contracts / Subscriptions which has "Date of Next Invoice" is equal to Start date on wizard.

    Print Contract Report

    Contract PDF Report

    Customized report of contract with full details and you can send to you customer. :)
