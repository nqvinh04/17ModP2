Work order/ Job Order Inspection for Project Construction Contracting Odoo Apps

Job Inspection for Project Costing Odoo apps allow you to have inspection feature on your Job Orders for Construction and Job Contracting Projects. After installing this module user can create job inspection for job order of project costing and able to add inspection result and value. This Odoo module also provide configuration option to add inspection result and values which can be select when inspection process conducted on job order of costing.

Features

Prints Report

User are allowed to see job inspection in report.

Job Cost and Project

User can see smart button on both project and Job Cost Sheet.

Create job inspection

User are allowed to create job inspection.

Inspection type and Inspection Result

User can create different inspection type and inspection result.

User Settings

In this app you have two type of user Inspection Manager and Inspection User


View Of Inspection

Under the menu of job inspection you can find the details of job inspection where you can enter details about the inspection and its results.



Process Of Inspection

After enter the details about the inspection and its result manager can chnge stage of the instructions to confirm,process,complete or cancel.


After confirming the inspection manager can process it.


After processing it manager will complete the inspection or can cancel at any stage.


After processing it manager will complete the inspection or can cancel at any stage.


After cancel the inspection will be in draft state if again manager want to process it he/she can do.


Smart Buttons

You can redrict to related inspection of paticular project and job cost sheet via smart button on project and job cost sheet.
