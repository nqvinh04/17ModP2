Car Repair App with Material Requisition Extension

Purchase Order & Stock Picking with Reference of Car Repair Request

This app allows your team to set a car repair reference on created internal picking and purchase orders from material requisition. It also set a job order reference on created internal picking from material requisition.

Main Features:

  • System pass car repair reference from material requisition into created internal pickings and purchase order.
  • Also pass job order / work order reference from material requisition into created internal pickings.
  • Please check the dependent app(s) for more details.
  • For more details please see the video and check below screenshots.
