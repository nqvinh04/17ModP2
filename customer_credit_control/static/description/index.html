Customer Credit Control

Amount Credit Control, Days Credit Control, Tolerance and Warnings.

This "Customer Credit Control " application allows user / Business to have complete control over credit sales done to the customers. These configurations can be done individually or Universally. Once configuration is done, system will take care of all the processes. Everything is automated and User / Business does not have to take care of anything manually. This application also has tolerance if user / business wants to let go credit limit check up to a certain level. If User / Business want to set warning, It also can be done in this application.

Customer Configuration

In customer form, this application adds credit control configurations. These configurations will apply to that particular customer only. Here, user / admin can configure credit control amount, credit control days, tolerance and warning email limits.

Credit Control Warning

This is a credit warning message, this will pop up when the user confirms the sale order but the customer has crossed the assigned credit limit. This warning message stops the sale order from getting confirmed.

Warning Email`

When the sale order gets confirmed and credit control is configured, system checks if customer due amount or credit due days have crossed their assigned limits. If does then system creates one credit control warning record. System has a cron to send warning emails that processes once a day right now but can be configured as per business need.

Universal Configuration

This is universal customer credit control. Fields are similar as customer’s form credit control. The only difference is customer credit control configured here will apply to all the customers in the system. except those who have credit control configured individually. Here you can see a checkbox named “Force Universal Credit Control”. It allows the overwriting of universal credit control over individual credit control. If it is checked true, the customer's individual credit control will be ignored and universal credit control will apply on them like all others.
