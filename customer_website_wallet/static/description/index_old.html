Website Wallet for Customers

This app provide wallet feature to your website where customer can add and withdraw from wallet for shopping.

Main Features:

  • Allow your customer of website to add money into wallet.
  • Allow your customer to view all wallet transaction he/she has made for add money and spend money of wallet.
  • Full history of wallet transactions.
  • Customer can do shopping from your website using wallet balance.
  • View of wallet balance on customer my account portal page.
  • Wallet payment on ecommerce page as payment gateway and we have created separate payment acquire for wallet.
  • Wallet transaction management in backend.
  • Allow you to add multiple wallet recharge products in system so customer can use that recharge to add money.
  • Email notification when money add into wallet or remove from wallet.
  • Warning message if balance is not enough in wallet but customer try to use wallet payment for shopping.
  • Fully integrated with Odoo Shop module.
  • Wallet transactions form and list view for purpose of reporting.
  • Customer can get benefit of using wallet.
  • Wallet is payment acquire so work with odoo standard shopping.
