Request for Information Construction and Contracting Business Odoo Apps

Job Costing Request for Information Odoo apps used to manage "Request for Information" for Construction and Contracting Business. Your customer/supplier can submit request online from your website and get details about the Project Job costing management.

Features

Creates Request for Information.

Users are allowed to create request for information from website.

Allows to Give Customer Rating

Users are allowed to give rating for the information in the website.

Send Message to the Customer.

You can send message to the customer with attachment.

User can Close the Information.

User can close the information at any point of time.

Specify the RFI Type and RFI Team.

User are allowed to create RFI type and RFI team.

Specify the RFI Stage and Category.

User are allowed to create the RFI stage and RFI category.

Menu Of Request For Information In Website

In webpage you can see menu for request for information(RFI).

Create Request For Information From Website

You can create request for information from website.You can add information about Subject of RFI, Name, Category and many more information.

Thank You Message

After submitting the information you will land up to Thank you page.

Request For Information Form View

You can create/see request for information Goto Job Costing > Request For Information > Request For Information.Here you can see form view of created request for information.

Timesheet View.

RFI Survey View

RFI Answer View.

Request For Information Tree View

Here you can see created all request for information.

Request For Information On My Account

You can see created request for information of login user from my account.

Request For Information List

After clicking on your request for information from my account, You can see all request of login user.

View Request For Information

For detail view of request for information you can click on any request for information.

Send Message and Attachments From Website

User can send message and attachments from here.Admin or Technician can communicate with customer using chatter.

Thank You Message

After submitting message and attachment you will land up to Thank you page and message and attachment will be attached to request for information.

Customer Rating Button

After click on customer rating button, It will redirect to rating page.

Website Customer Rating

Here you can write review for request for information.

Customer Rating At Backend

Request For Information Integration with Survey

Click on survey button to view created survey of request for information.

Close Request For Information

Click on set to close button to close request for information. After closing request, Stage will be changed to closed and current date will be set as closed date.
