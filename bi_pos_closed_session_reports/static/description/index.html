POS Cash Register End Of Session Z-Report Odoo Apps

POS Cash register End Of Session Z-Report Odoo Apps support both community as well as enterprise version. This Point of sales module will allow user to print POS Current and Closed Session Reports which consist total transaction and sales summary report at End of shift and total sales summary of session. You can generate report for Current and Closed sessions in PDF format and This odoo apps is POS-BOX Compatible.

Features

POS Configuration for Session Report

Under POS configuration user have options to print receipt or report with product wise details, category wise details, pricelist wise details, payment wise details and customer wise details.

Print Current Session Report from POS Screen

User can print only current session report directly from POS screen in PDF format.

Select Report Type from POS Configuration

User have options to print report with type of PDF, Receipt or both.

Posted Session Report from POS

User can easily print posted session report from POS screen.

Print Session Report from POS Session

User can print and view session wise report from POS session in PDF format.

Posted Session Report

Using Posted Session Report, user can generate report for Closed sessions.

POS Session Report Configuration

Under point of sale configuration user can tick on 'Enable Session Report' checkbox for print current session report directly from POS screen.

User have option to report type as 'PDF', 'Receipt' and 'Both' to print from POS session.

User have another options to print receipt or report with product wise details, category wise details, pricelist wise details, payment wise details and customer wise details.

Print Session Report from POS Screen

User can click on 'Print Session Report' button on point of sale screen for print only current session report.

Print Report Wizard

When user configure report type as 'Both' then always ask for print report as 'Receipt' or 'PDF' and click on 'Print' button.

Generated Current Session Receipt from POS Screen

POS Session Report Configuration

User can tick on 'Allow Posted Session Report' checkbox for print posted session report directly from POS screen.

Posted Session Report from POS Screen

User can click on 'Posted Session Report' button on point of sale screen for print any posted session report.

Posted Session Report Wizard

User can select any of the posted session from drop down list and click on 'Print' button.

Generated Posted Session Report from POS Screen


Print Session Report from POS Session

User can print and view session wise report from POS session.

Generated Session Report from POS Session

Posted Session Report

User can select single or multiple POS session(s) to generate report of posted or closed sessions.
