Construction Work Package for Job Costing and Contracting Odoo Apps

Work Package for Construction and Contracting Odoo Apps allow you to create work package planning for job costing. This created planing for work package for job costing you can send to your customers easily for your construction and contracting business. Work package can be creates with multiple job orders and process for construction projects and this planning can be easily send to customers to check the package costing for the Construction projects.
