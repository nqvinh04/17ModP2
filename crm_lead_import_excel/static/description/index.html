CRM Lead / Pipeline Import from Excel File

Import CRM Leads in Odoo

This module allows you to import Leads / Pipeline from excel file.

Main Features:
  • Allow you to Import Leads/Pipeline from excel file
  • Make sure you have xlrd Python module installed.
  • See below fields list which are allowed in import file.
  • Sample Excel is available inside the module.
  • For more details please watch the video and see the screenshots below.
Menus:
  • CRM
    • CRM/Import
      • CRM/Import/Import Leads/Pipeline
Following fields list which can be imported in Lead.
  • Name : Name of Leads/Pipeline.
  • Customer : Name of Customer.
  • Company Name : Partner name.
  • City : City name.
  • Country : Country name.
  • State : State name.
  • Website : Website.
  • Salesperson : Salesperson name.
  • Sales Team : Sales Team name.
  • Contact Name : Contact name.
  • Email : Email Address.
  • Job Position : Job Position.
  • Phone : Phone Number.
  • Mobile : Customer Mobile Number Enter In Mobile Field.
  • Tags : Tags Like Product , Information etc...
  • Internal Notes : Description of Leads/Pipeline.
  • Campaign : Campaign Like Sale , Christmas Special etc...
  • Medium : Medium Like Email , Phone etc...
  • Source : Source Like Facebook , Twitter etc...
  • Referred By : A Person Come With Which Person Reference Is Enter In Referred By.
  • Probability : Probability.


    • Sample Excel is available inside module.
