Calendar Scheduler by Assignee on Working Hours

New menu to schedule calendar meetings / events by assignee using FullCalendar Scheduler vertical resource view and highlights / restricts the working hours in the view

Features:

  • New menu to schedule meetings by assignee in vertical resource view
  • See meetings for each assignee by day view and week view
  • Create meetings for each assignee
  • Drag and drop meetings from one assignee to another
  • Filter calendar resources by assignee
  • Highlights or restricts the working hours in the view

Calendar Configuration

Go to Calendar > Configuration > Settings. Set the FullCalendar premium license key if needed and set the desired working hours for calendar view. User must have Administration / Settings group access and must be in developer mode.

Assignee Resource Configuration

Configuration can be found in Settings > Users & Companies > Users. Then setup the users which will be a meeting assignee
