Request for Quotation / Purchase Order from CRM Pipeline / Opportunity

Create RFQ / Purchase Order from Lead / Opportunity / Pipeline in Odoo

CRM Lead / Opportunity / Pipeline with Purchase Order Creation

This app allows your sales team/users to create a purchase order/request for quotation from CRM lead/opportunity/pipeline as shown in the below screenshots.

Main Features:

  • Allow your user to create an RFQ / Purchase Order from CRM Opportunity / Pipeline for selected vendors.
  • Allow your user to see all created RFQ(s)/Purchase Order(s) from the smart button on the opportunity/pipeline formRFQ will be created based on Vendors selected on the product line on the opportunity form.
    (All product lines will be grouped by the vendor and multiple RFQ will be created based on selected vendors.).
  • Opportunity CRM reference will be stored on RFQ / Purchase Order form of Odoo standard.
  • For more details please check the below screenshots and watch the video.
