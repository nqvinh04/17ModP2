Create Material Requisitions from Tailoring Cloth Request

Material Requisitions for Work Order Request for Tailoring

Material Request from Cloth Request & Work Order Request Odoo

This app allows you to create material requisitions from the tailoring of cloth request & tailoring cloth work order request form as shown in below screenshots.


Main Features:
  • Allows your cloth & project team to create material requisitions from the tailor cloth request form and work order (Task).
  • Using this module your tailoring team can create requests for materials directly from request or from work order they are working. They can request material which will be required to do that job.
  • For more details please see the video and check below screenshots.

Edition Compatibility:
